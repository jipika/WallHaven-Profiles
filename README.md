# WallHaven Profiles

规则仓库 - 支持壁纸、媒体、动漫解析规则

---

## 🇨🇳 中文

### 简介
WallHaven Profiles 是 WallHaven 壁纸应用的自定义规则仓库。只需在 App 中填入此仓库地址，即可自动同步所有规则。

### 使用方法
1. 打开 WallHaven App
2. 进入「设置」→「规则仓库」
3. 输入仓库地址：`https://github.com/jipika/WallHaven-Profiles`
4. 点击「保存并同步」

### 仓库结构
```
WallHaven-Profiles/
├── index.json          # 规则索引
├── wallhaven.json      # WallHaven 壁纸规则
└── anime/              # 动漫规则
    ├── index.json      # 动漫规则索引
    ├── age.json         # AGE 动漫
    ├── 7sefun.json      # 7sefun 动漫
    ├── dm84.json        # DM84 动漫
    └── gimy.json        # Gimy 动漫
```

### 添加自定义规则
1. Fork 此仓库
2. 添加新的 JSON 规则文件
3. 更新 `index.json` 或 `anime/index.json`
4. 提交 Pull Request

## 🇺🇸 English

### Overview
WallHaven Profiles is a custom rule repository for the WallHaven wallpaper app. Simply enter this repository URL in the app to automatically sync all rules.

### How to Use
1. Open WallHaven App
2. Go to Settings → Rule Repository
3. Enter repository URL: `https://github.com/jipika/WallHaven-Profiles`
4. Click "Save and Sync"

### Repository Structure
```
WallHaven-Profiles/
├── index.json          # Rule index
├── wallhaven.json      # WallHaven wallpaper rule
└── anime/              # Anime rules
    ├── index.json      # Anime rule index
    ├── age.json         # AGE Anime
    ├── 7sefun.json      # 7sefun Anime
    ├── dm84.json        # DM84 Anime
    └── gimy.json        # Gimy Anime
```

### Adding Custom Rules
1. Fork this repository
2. Add new JSON rule files
3. Update `index.json` or `anime/index.json`
4. Submit a Pull Request

## 🇯🇵 日本語

### 概要
WallHaven Profiles は、WallHaven 壁纸アプリのカスタムルールリポジトリです。アプリでこのリポジトリURLを入力するだけで、すべてのルールが自動的に同期されます。

### 使用方法
1. WallHaven アプリを開く
2. 「設定」→「ルールリポジトリ」に移動
3. リポジトリURLを入力：`https://github.com/jipika/WallHaven-Profiles`
4. 「保存して同期」をクリック

### リポジトリ構造
```
WallHaven-Profiles/
├── index.json          # ルールインデックス
├── wallhaven.json      # WallHaven 壁纸ルール
└── anime/              # アニメルール
    ├── index.json      # アニメルールインデックス
    ├── age.json         # AGE アニメ
    ├── 7sefun.json      # 7sefun アニメ
    ├── dm84.json        # DM84 アニメ
    └── gimy.json        # Gimy アニメ
```

### カスタムルールの追加
1. このリポジトリを Fork
2. 新しいJSONルールファイルを追加
3. `index.json` または `anime/index.json` を更新
4. Pull Request を提交
